showUserData(request, uid)

source code 

Show User Data is the function that renders the public user URL.

This could be an empty webpage or an alert page that informs that an user us un missing state. If anyone has installed the physica web beacon official application will be notified if detects the beacon.

Parameters:
  • request - Request petition that is handled by Django
  • uid - The uid is the url identifier that represents the user id.

    Example:

    • url: /p/5/ to get the information about user number 5.
