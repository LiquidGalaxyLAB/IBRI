Class Clients

source code

The Clients models represents a user instance. This data could be useful for the emergency squad to know the user important data as for example the phone number, diseases, etc. The most of the data is encrypted in the database using AES256 CBC algorithm. The key to crypt or decrypt the data is stored in the settings SKEY variable.

Requires: Has a SKEY stablished in the database
