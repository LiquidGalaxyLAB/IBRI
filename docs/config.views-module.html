downloadfile(request, m, r)

source code 

Download File returns an KML filename as petition. The files have the following format: IBRI[m]R[r].kml where the m is the mission id and the r is the route id.

Parameters:
  • request - Request petition that is handled by Django
  • m - Represents the mission id
  • r - Represents the route id @see FileWrapper @see Content-Disposition
